# Page Content Source Map

## Purpose

This document maps approved internal source material to each website page before final public copy is written.

It is a content planning map only. It does not provide final website copy, pricing, case studies, service guarantees, compliance claims, stack claims, contact form behavior, analytics wording, or deployment instructions.

## Current content status

The files in `content/pages/` are placeholders. They are not approved public copy.

Use this source map when replacing placeholders with drafted page content. Public copy still needs review against the relevant source docs and guardrails before publication.
