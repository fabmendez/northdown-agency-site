# Requirements

## Goal

Scaffold the project for the Northdown Computers AI Systems & Automation website and operating base.

## Current acceptance criteria

- Project folder exists at `northdown-agency-site/`.
- Template structure is copied from `templates/codex-agentic-engineering-template/`.
- Project-brain docs exist for charter, requirements, architecture, decisions, source references, and handoff.
- No website implementation code is added.
- No dependencies are installed.
- No dev server is started.

## Future product requirements

To be defined in approved tasks before implementation. Do not infer services, pricing, copy, integrations, or workflows without explicit source material.

## Constraints

- Keep changes documentation-only until an approved build task exists.
- Use local source references before external docs.
- Ask before adding dependencies, integrations, analytics, forms, payments, or authentication.
