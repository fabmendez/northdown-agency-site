# Project Charter

## Purpose

Create the website and operating base for Northdown Computers AI Systems & Automation.

The site should eventually explain the business clearly, support client trust, and become the base for future operational workflows. This scaffold does not implement the website yet.

## Target users

- Local businesses evaluating AI systems, automation, and technical support from Northdown Computers AI Systems & Automation.
- Internal operators using the project-brain to plan and build the business platform safely.
- Future agents working from explicit tasks and source truth.

## Definition of success

- Project-brain docs define the goal, requirements, constraints, and architecture before code exists.
- Future implementation tasks can be created without guessing business rules.
- The first build task can proceed from documented scope, approved stack choices, and validation expectations.
